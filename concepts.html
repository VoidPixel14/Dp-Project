Structure Tags

<h1>
Defines a top-level heading. Use <h1> to <h6> for headings of different importance.
<p>
Defines a paragraph. Automatically creates spacing before and after the text.
<br>
Inserts a single line break. Commonly used in addresses and poems.
<ul>
Creates an unordered (bulleted) list. Use <li> for each list item.
<ol>
Creates an ordered (numbered) list. Used for sequences and steps.

Form Elements

<form>
Defines a form for user input. Contains input elements like text fields and buttons.
<input>
Defines an input field where users can type data.
<button>
Represents a clickable button used to submit forms or trigger actions.

Media and Links

<a>
Creates a hyperlink. Use the href attribute to define the link destination.
<img>
Embeds an image into the page. Requires src and alt attributes.
<video>
Used to embed video content with optional controls.

Text and Formatting

<b>
Makes text bold without implying importance.
<i>
Italicizes text, often for alternate voice or mood.
<mark>
Highlights text with a yellow background.
<del>
Represents deleted text, typically shown with a line through it.
<u>
Underlines text.
<code>
Displays text in a monospaced font, used for code snippets.

Table Elements

<table>
Defines a table to organize data in rows and columns.
<tr>
Represents a table row.
<th>
Defines a header cell in a table.
<td>
Defines a standard data cell in a table.

Semantic Elements

<section>
Defines a thematic grouping of content, typically with a heading.
<article>
Defines self-contained content that could be independently distributed.
<aside>
Defines content aside from the main content, like sidebars or callouts.
<nav>
Defines a section with navigation links.

Meta and Head Tags

<meta>
Provides metadata about the document, like character encoding and description.
<title>
Defines the title of the document shown in the browser tab.
<link>
Links external resources like stylesheets.
<style>
Embeds CSS styles inside the document head.
